  <name>Task 2: Create root package.json with dev command</name>
  <files>package.json</files>
  <action>
Create root-level `package.json` that:
1. Defines a `dev` script using `concurrently` to start both backend and frontend
2. Backend: `cd packages/clerk && uv run clerk web --port 8000`
3. Frontend: `cd apps/website && npm run dev`
4. Includes concurrently as devDependency
5. Uses different colors/prefixes for backend vs frontend output

Per SETUP-03: Must enable "single command dev environment start".

Structure:
```json
{
  "name": "openclerk-workspace",
  "private": true,
  "scripts": {
    "dev": "concurrently \"npm:dev:backend\" \"npm:dev:frontend\" --names \"API,UI\" --prefix-colors \"blue,green\"",
    "dev:backend": "cd packages/clerk && uv run clerk web --port 8000",
    "dev:frontend": "cd apps/website && npm run dev"
  },
  "devDependencies": {
    "concurrently": "^8.2.0"
  }
}
```
